A Simple Naïve Pathtracer I wrote:

Rendered with 512 samples at a resolution of 1080x810

I have written a path tracer implemented CUDA C(++) just as a project to test my programming skills or just for fun and interest. Though, there isn't support for actual meshes yet, it's just spheres (and yes, the ground is a massive sphere too), but there is support for reflections, fuzzy reflections, refraction and global illumination. I am planning to release the code in github soon.

Things I am going to add in the future:

  • Cubes, planes, and mesh support
  • Light dispersion in transparent objects
  • Proper system for rendering video
